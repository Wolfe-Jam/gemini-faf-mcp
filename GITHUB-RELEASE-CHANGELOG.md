# 🚀 What's New in v2.1.2: The Universal Context Update

This release solidifies **gemini-faf-mcp** as the universal standard for AI project context, unifying the industry's siloed formats into a single source of truth.

## 🧬 Universal Project DNA
*   **The "Big Three" Bridge:** Native support for unifying `CLAUDE.md`, `GEMINI.md`, and `AGENTS.md` into one IANA-registered `.faf` file.
*   **SEO Optimized:** Improved discovery on PyPI and GitHub with high-traffic keywords for context management and AI agents.

## 🚀 Key Features
*   **Auto-Detect v2:** Enhanced `faf_auto` logic for scanning project manifests (`pyproject.toml`, `package.json`, `Cargo.toml`, `go.mod`) and generating 100% accurate project DNA.
*   **Trophy-Tier Reference:** Access to 15 different 100% "Trophy" scored project models via the `faf_model` tool.
*   **IANA Compliance:** Full alignment with the latest `application/vnd.faf+yaml` specification.

## 🏛️ Strategic Foundations
*   **Proposal Ready:** Includes a formal feature proposal for native FAF integration in the Gemini CLI.
*   **Sponsor Program v1.0:** Introduction of the FAF Foundation Sponsor & Badge program for strategic partners.
*   **Outreach Templates:** Professional communication kit for Anthropic, Google, and OpenAI partnerships.

## 🛠️ Internal Improvements
*   Updated `pyproject.toml` with refined classifiers and keywords.
*   New `README.md` with high-impact "Problem/Solution" messaging and comparison table.
*   Organized project structure for scale and secure context management.
